About

JSR-303 Validators to validate a daterange and other constraints.

Constraints

Following constraints are available for now:

AllowedIntegers
Validate an integer value against a list of integers
AllowedStrings
Validate a string against a list of strings
DateRange
Validate the interval of a begin and end date
Nested
Validates a class constraint or a collection of it
Future
Check if a date is in the future or if required today or in the future

Dependencies

The only dependency is joda-time 1.2.1. This version is used to be compatible with the version delivered by Oracle in their Weblogic 10.3.x.
